# T-REX — циферблат для Amazfit Balance 2

Адаптация циферблата **T-REX** (тактический стиль) под **Amazfit Balance 2** — круглый экран 480×480, Zepp OS 5. На главном экране: крупное время, дата, шаги, калории, заряд, будильник, таймер и всплывающий прогноз погоды.

## Установка

Циферблат ставится через **режим разработчика Zepp** сканированием QR-кода — без регистрации appId и без смены аккаунта/региона (appId вне dev-диапазона, пакет хостится здесь, на GitHub Pages).

<p align="center">
  <img src="assets/qr.png" width="200" alt="QR-код для установки">
</p>

1. Открой приложение **Zepp** на телефоне (часы Amazfit Balance 2 подключены).
2. Включи **режим разработчика** (Developer Mode) в Zepp.
3. В режиме разработчика **отсканируй QR-код** выше — либо открой сохранённый файл QR через «OPEN».
4. Zepp скачает пакет с GitHub и установит циферблат на часы по Bluetooth — он появится в списке циферблатов.

**Прямая ссылка на пакет:** [`T-REX_Kirill_Balance2.zpk`](T-REX_Kirill_Balance2.zpk) (348 КБ)

QR кодирует ссылку: `zpkd1://flash797.github.io/T-REX_Kirill_Balance2.zpk`

## Что показывает циферблат

- Крупное цифровое время (ЧЧ:ММ) и дата
- Будильник и таймер
- Шаги, калории, заряд батареи
- Всплывающий прогноз погоды (по тапу)

## Как устроена адаптация

Циферблат адаптирован под Balance 2 вручную, скриптами (не через редактор — редактор даёт двойной ресайз и портит чёткость):

- **Кратность ширины 4** — движок Zepp требует ширину спрайта кратной 4 (выравнивание текстуры). Фон расширен до 496×480 (overscan), что убирает горизонтальные полосы.
- **Pixel-perfect глифы** — цифры и тексты берутся из оригинала байт-в-байт, без ресайза → остаются резкими.
- **Попап погоды** — координаты столбцов прогноза домасштабированы ×1.03 под ширину 480, чтобы не съезжали вправо.

## Лицензия и авторство

Дизайн циферблата **T-REX** принадлежит его автору (сторонний циферблат для Zepp OS). Этот репозиторий — только техническая адаптация под Amazfit Balance 2 и личный хостинг пакета для установки на свои часы. Не для коммерческого использования.
